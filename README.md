# 🛠️ Work Tools

Eine Sammlung spezialisierter, browserbasierter Werkzeuge für die Trinkwasserhygiene, Dokumentation und den technischen Außendienst.

Diese Tools sind für die mobile Nutzung optimiert und ermöglichen eine schnelle, rechtssichere Erfassung von Daten direkt vor Ort.

## 💧 Hygiene & Protokollierung

In diesem Bereich finden sich Werkzeuge zur Dokumentation von Spülmaßnahmen gemäß den geltenden Richtlinien (z. B. VDI 6023).

### [Thermische Desinfektion](/spuelprotokoll/)
* **Fokus:** Einzelschutzmaßnahme / Sanierung.
* **Einsatz:** Dokumentation von Hochtemperatur-Spülungen nach Grenzwertüberschreitungen (z. B. Legionellenbefund).
* **Funktionen:** Erfassung von Vorlauf-/Rücklauftemperaturen und Einwirkzeiten für spezifische Objekte.

### [Spülprotokoll Anlagen (Prävention)](/spuelprotokoll-anlagen/)
* **Fokus:** Bestimmungsgemäßer Betrieb / Legionellenvorbeugung.
* **Einsatz:** Dokumentation des regelmäßigen, vorsorglichen Wasseraustauschs zur Vermeidung von Stagnation.
* **Funktionen:** Protokollierung von Routine-Spülplänen für komplexe Liegenschaften.

## 📅 Organisation & Labor

Werkzeuge zur Koordination von Prüfintervallen und der Probenahme-Logistik.

### [Terminverwaltung Labor](/Terminverwaltung-labor/)
* **Fokus:** Kunden- und Objektmanagement.
* **Einsatz:** Planung von wiederkehrenden Laboruntersuchungen und Probenahmen.
* **Struktur:** * Verwaltung von Kundenstämmen.
    * Zuordnung mehrerer Objekte pro Kunde.
    * Überwachung von Fristen und Intervallen.

---

## 🚀 Features & Technik

* **Offline-First:** Optimiert für den Einsatz in Kellern oder Technikräumen mit schlechtem Empfang.
* **Responsive Design:** Volle Funktionalität auf Smartphones, Tablets und Laptops.
* **Datenschutz:** Die Verarbeitung der Daten erfolgt lokal im Browser (kein Cloud-Zwang für sensible Objektdaten).

## 🛠 Installation / Nutzung

Die Tools sind direkt über die GitHub Pages erreichbar. Einfach das gewünschte Tool in der Übersicht auswählen und im Browser nutzen. Für den schnellen Zugriff empfiehlt es sich, das jeweilige Tool als "Web-App" (Add to Home Screen) auf dem Mobilgerät zu speichern.
